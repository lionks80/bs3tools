bs3paging

Bootstrap3 paging jQuery Plugin.

Example Data

The following data is used in the tutorial.

// Example Data
var sampleData = {
	"start" : 60,
	"limit" : 20,
	"totalCount" : 318,
	"displayCount" : 9,
	"pagingCss" : [ "pagination" ],
	"trigger" : function(page) {
		alert(page);
	},
	"useEndBtn" : true,
	"useNextBtn" : true
};

Tutorial

Create Grid & Data Bind

$(function() {

	$('#tutorial_paging').bs3paging(sampleData);
	
	$('#btn_tutorial_getCurrentPage').click(function() {
		alert($('#tutorial_paging').bs3paging('getCurrentPage'));
	});
	
});

Tutorial Example

check point

  • getCurrentPage: get current page
  • when click a page number trigger fired.

Options

Name Type Description
start integer 현재 게시물 시작 번호
limit integer 페이지 출력 수
totalCount integer 전체 게시물 수
displayCount integer 페이징 버튼 표시 수
pagingCss string array 내부에 생성되는 ul 태크에 적용할 css 클래스들
trigger function(page) 페이징 버튼들을 클릭했을 시 동작
useEndBtn boolean 처음으로, 끝으로 버튼 사용 여부
useNextBtn boolean 이전 또는 다음 페이지로 버튼 사용 여부
